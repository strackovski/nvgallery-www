Quick Guide

1. What you need

jQuery

nvGallery files (jquery.nvgallery.js, nvgallery.css)

A collection of icons called FontAwesome (included in the download)

2. Creating an array of images

In your Javascript file, create an array of object literals with a defined path property and an optional description property.

var myPictures = [
    {path:'images/rocks.png', description:'Just a picture.'},
    {path:'images/paws.png', description:'Another photo I took.'}
];

3. Make a div element

Optionally you can set its width (height is calculated dynamically). Otherwise the gallery will take 100% of width.

HTML:

<div class="wrap"><div>

CSS:

.wrap{
    width:500px;
}
Width can also be defined on a div's parent element.

4. Call the plugin

Inside your document.ready() event, call the plugin on the created div element and pass the images array as an argument. 

$('.wrap').nvgallery(myPictures);

Detailed Guide

1. What you need

jQuery

nvGallery files (jquery.nvgallery.js, nvgallery.css)

A collection of icons called FontAwesome (included in the download)

Transfer the files to your project and reference them in the HTML document like so: 

<!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" src="jquery-1.11.0.js"></script>
        <script type="text/javascript" src="jquery.nvgallery.js"></script>
        <link rel="stylesheet" href="nvgallery.css"></script>
    </head>
    <body>
    ...
    </body>
</html>
Instead of downloading jQuery, you can also include a link of the file hosted online:
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>

2. Creating an array of images

In your Javascript file, create an array of object literals with a defined path property and an optional description property. 

var myPictures = [
    {path:'images/rocks.png', description:'Just a picture.'},
    {path:'images/paws.png', description:'Another photo I took.'}
];

3. Make a div element

Optionally you can set its width (height is calculated dynamically). Otherwise the gallery will take 100% of width.

HTML:

<div class="wrap"><div>

CSS:

.wrap{
    width:500px;
}
Width can also be defined on a div's parent element.
nvGallery is responsive so you can also express the width in percents.

4. Call the plugin

Inside your document.ready() event, call the plugin on the created div element and pass the images array as an argument. 

$('.wrap').nvgallery(myPictures);

With no options object passed, the defaults take effect, namely:

height: '0.5625'

arrows style: border radius of 10% in outer corners with white background

container style: dark background with no border

autoplay: false

5. The result


6. Options

Options allow you to adjust the behavior and look of the plugin. If no options are provided, the defaults will take effect.

type: number
default: 3
possible options: any number

Usage 
$('ul').itemSlider({showItems:6});
Details

Sets the number of items to show in a row.

Available options are:

  • height

  • interval

  • disableMenuBar

  • hideMenuBar

  • autoplay

  • hideNav

  • disableNav

  • leftArrowCss

  • rightArrowCss

  • containerCss

  • barCss

  • fadeDuration


An example of a gallery with width set to 400px, height: 1.2, autoplay enabled, a 2px light gray border, a white bottom bar and round navigation arrows.


Call the plugin with options:

$('div').nvgallery(
    {barCss:{background:'#fff', color:'#000'},
    autoplay:true,
    containerCss:{border:'2px solid #999', background:'#fff'},
    rightArrowCss:{borderRadius:'50%', right:'10px', background:'#111', color:'#fff'},
    leftArrowCss: {borderRadius:'50%', left:'10px', background:'#111', color:'#fff'}
});
